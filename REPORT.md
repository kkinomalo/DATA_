# Heart Attack Risk Prediction Dataset을 활용한 1D CNN 기반 심장마비 위험 예측 보고서

## 1. 연구 주제

본 프로젝트의 주제는 환자의 건강 정보를 활용하여 심장마비 위험 여부를 예측하는 1D CNN 기반 분류 모델을 구현하는 것이다.

## 2. 데이터셋 소개

본 프로젝트에서는 Kaggle의 Heart Attack Risk Prediction Dataset을 사용하였다. 해당 데이터셋은 환자의 건강 상태, 생활 습관, 심혈관 관련 지표 등을 포함하는 표 형식의 데이터이다. 목표 변수는 심장마비 위험 여부를 나타내며, 모델은 이를 0 또는 1로 분류한다.

## 3. CNN 사용 이유

일반적인 CNN은 이미지 데이터 분석에 자주 사용된다. 그러나 이 프로젝트의 데이터셋은 이미지가 아닌 표 형식 데이터이므로 2D CNN을 그대로 사용하는 것은 적절하지 않다. 따라서 각 환자의 특성 벡터를 1차원 배열로 보고, 1D CNN을 사용하였다.

1D CNN은 여러 입력 특성 사이의 지역적인 패턴을 학습할 수 있으며, 표 데이터의 특성 벡터를 `(샘플 수, 특성 수, 1)` 형태로 변환하면 Conv1D 계층에 입력할 수 있다.

## 4. 데이터 전처리

전처리 과정은 다음과 같이 진행하였다.

1. 컬럼명을 소문자와 밑줄 형식으로 정리하였다.
2. ID처럼 학습에 불필요한 컬럼을 제거하였다.
3. 혈압처럼 `120/80` 형태로 저장된 문자열 컬럼은 수축기 혈압과 이완기 혈압으로 분리하였다.
4. 숫자형 컬럼은 결측치를 중앙값으로 채우고 표준화를 적용하였다.
5. 범주형 컬럼은 결측치를 최빈값으로 채우고 원-핫 인코딩을 적용하였다.
6. 타깃 변수는 LabelEncoder를 사용하여 숫자 라벨로 변환하였다.
7. 데이터를 학습용, 검증용, 테스트용으로 분리하였다.
8. 1D CNN에 입력하기 위해 데이터 형태를 `(샘플 수, 특성 수, 1)`로 변환하였다.

## 5. 모델 구조

본 프로젝트에서 사용한 1D CNN 모델은 다음과 같이 구성하였다.

- Conv1D 계층
- Batch Normalization
- MaxPooling1D
- Conv1D 계층
- Batch Normalization
- MaxPooling1D
- Conv1D 계층
- Batch Normalization
- GlobalAveragePooling1D
- Dense 계층
- Dropout
- Softmax 출력층

Conv1D 계층은 환자 건강 지표 사이의 패턴을 추출하며, Dense 계층은 추출된 특징을 바탕으로 최종 클래스를 예측한다.

## 6. 학습 방법

손실 함수는 `sparse_categorical_crossentropy`를 사용하였고, 최적화 알고리즘은 Adam Optimizer를 사용하였다. 또한 검증 손실이 더 이상 개선되지 않을 경우 학습을 조기 종료하기 위해 EarlyStopping을 적용하였다.

데이터 불균형을 완화하기 위해 클래스 가중치를 계산하여 학습 과정에 반영하였다.

## 7. 평가 방법

모델 성능 평가는 다음 지표를 사용하였다.

- 테스트 정확도
- Classification Report
- Confusion Matrix
- Loss Graph
- Accuracy Graph

## 8. 실행 결과 예시

실행 결과 예시 Confusion Matrix는 다음과 같다.

```text
              Predicted 0   Predicted 1
True 0              700           112
True 1              220           368
```

전체 테스트 데이터는 1400개이며, 이 중 올바르게 예측한 개수는 700 + 368 = 1068개이다.

따라서 예시 기준 테스트 정확도는 다음과 같다.

```text
1068 / 1400 = 0.7628
```

즉, 약 76.3%의 정확도를 보였다.

## 9. 결과 분석

Confusion Matrix 분석 결과, 실제 클래스 0인 데이터 812개 중 700개를 올바르게 예측하였고, 112개는 클래스 1로 잘못 예측하였다. 실제 클래스 1인 데이터 588개 중 368개를 올바르게 예측하였고, 220개는 클래스 0으로 잘못 예측하였다.

이를 통해 모델은 클래스 0에 대한 예측 성능이 상대적으로 높지만, 클래스 1에 대한 예측 성능은 다소 낮다는 것을 확인할 수 있다. 특히 의료 위험 예측 문제에서는 위험이 있는 데이터를 위험이 없다고 예측하는 경우가 중요하므로, 클래스 1에 대한 재현율을 높이는 추가 개선이 필요하다.

## 10. 한계점 및 개선 방향

본 프로젝트의 한계점은 다음과 같다.

첫째, 표 데이터에서는 CNN보다 Random Forest, XGBoost, MLP와 같은 모델이 더 좋은 성능을 보일 수 있다. 따라서 다른 머신러닝 모델과의 비교 실험이 필요하다.

둘째, 의료 데이터는 실제 진단에 직접 사용하기 어렵다. 실제 의료 분야에 적용하기 위해서는 전문가 검토와 추가 검증이 필요하다.

셋째, 클래스 1에 대한 예측 성능을 높이기 위해 데이터 불균형 처리, 하이퍼파라미터 튜닝, 모델 구조 개선 등이 필요하다.

## 11. 결론

본 프로젝트에서는 Heart Attack Risk Prediction Dataset을 사용하여 1D CNN 기반 심장마비 위험 예측 모델을 구현하였다. 이미지가 아닌 표 형식 데이터에 대해 1D CNN을 적용하기 위해 각 환자의 특성 벡터를 1차원 입력으로 변환하였다.

실험 결과, 모델은 약 76.3%의 테스트 정확도를 보였으며, 클래스별 예측 성능을 Confusion Matrix를 통해 분석할 수 있었다. 이를 통해 표 형식 의료 데이터에도 1D CNN을 적용할 수 있음을 확인하였다.
